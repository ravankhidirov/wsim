Wave simulation using EFIT method implemented with Numpy and Cupy (for cpu and gpu). Multi-GPU version is being developed using cupynumeric.

References:

Molero-Armenta, M., Iturrarán-Viveros, U., Aparicio, S., & Hernández, M. G. (2014). Optimized OpenCL implementation of the Elastodynamic Finite Integration Technique for viscoelastic media. Computer Physics Communications, 185(10), 2683–2696. https://doi.org/10.1016/j.cpc.2014.05.016
